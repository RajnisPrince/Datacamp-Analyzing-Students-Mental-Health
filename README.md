# Analyzing Students' Mental Health 🧠✈️

This repository contains my solution to the DataCamp project **"Analyzing Students' Mental Health"**. The project explores a 2018 study conducted by a Japanese international university to analyze how studying abroad impacts students' mental health. 

Using data analysis, this project investigates whether international students face a higher risk of mental health difficulties and evaluates how social connectedness and acculturative stress act as primary predictors of depression.

---

## 📊 Project Objectives
* Explore the dataset using basic descriptive statistics.
* Analyze the impact of length of stay on the mental health scores of international students.
* Determine if diagnostic scores (`PHQ-9`, `SCS`, `ASIS`) vary significantly based on demographic factors.

## 🛠️ Tech Stack & Skills
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Environment:** Jupyter Notebook / DataCamp Workspace
* **Key Concepts:** Data Wrangling, Exploratory Data Analysis (EDA), Descriptive Statistics, Aggregation

---

## 📈 Dataset Overview
The dataset includes diagnostic measurements from various mental health tests alongside demographic data of the students. The primary metrics used are:

| Variable | Description |
| :--- | :--- |
| `inter_dom` | Student type (International or Domestic) |
| `stay` | Current length of stay (in years) |
| `todep` | Total score of depression (measured via PHQ-9 test) |
| `tosc` | Total score of social connectedness (measured via SCS test) |
| `toas` | Total score of acculturative stress (measured via ASIS test) |

---

*(Tip: Replace these bullet points with your specific data insights once you finish running your final notebook! For example:)*
